Example

Get block hash by height

  /api/block-index/[:height]
  /api/block-index/1

This would return:

{"blockHash":"000009cb25c348b85b01819c52402adea224580263fbe9b915be8502c5220f82"}

which is the hash of the Genesis block (1 height)
